Fused QKV → 8 heads

A single Linear(d, 3d) projects the hidden state, then mx.split peels off Q, K, V and reshapes to [8, T, 48].

Causal mask — reveal across positions

Position i attends to itself and all j ≤ i. Future tokens are set to -inf before softmax — the lower triangle lights up as the model marches forward.

§ 03 · Implementation notes
MLX · bf16 · 400 LOC
N · 01

Tied embeddings

There is no separate LM head. GPT.__call__ ends with tok_emb.as_linear(x), projecting hidden states back to vocab using the input embedding's weights — saving ~19M parameters.

N · 02

Fused QKV

A single Linear(d, 3·d) with no bias produces Q, K, V at once. They're split along the last axis, then reshaped into eight heads of 48 dims.

N · 03

Causal mask

An upper-triangular boolean mask (mx.triu(…, k=1)) sends future positions to -inf before softmax — the only thing that makes attention auto-regressive.

N · 04

bf16 + lazy eval

Parameters live in bfloat16; the loss is cast to float32 for log-softmax stability. mx.compile fuses fwd/bwd/AdamW into one graph; mx.eval forces materialisation.
